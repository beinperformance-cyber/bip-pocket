# BIP Pocket

Prototype pédagogique de Progressive Web App (PWA).

## Fonctions
- écran adapté au smartphone ;
- ajout d'idées et de tâches ;
- stockage local sur l'appareil ;
- chronomètre ;
- fonctionnement hors connexion après la première ouverture ;
- installation sur l'écran d'accueil.

## Fichiers
- `index.html` : interface et logique principale ;
- `manifest.webmanifest` : nom, icône et comportement d'installation ;
- `sw.js` : cache hors ligne ;
- `icons/` : icônes de l'application.

## Publication
Publier le contenu du dossier sur un hébergement HTTPS statique, par exemple GitHub Pages.
